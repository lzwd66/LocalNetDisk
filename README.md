# LocalNetDisk
An extremely lite Flask web app to help you transfer files in your remote device.

## Function
- upload files
- download files
- download folders as zip files
- which your issues suggest!

## Usage
- download the code

- install Python and Flask framework
``` bash
pip install flask
```

- run this command in where app.py is
``` bash
python app.py
```

- run this command to get your ip
``` bash
ipconfig
```

- Then you can use any device in your LAN, or use other techs to visit the server with the URL '[your ip]:5000'. just enjoy~

## Advance Usage
You can modify many parameters in source code.
